# MediaVibe

A media player that aims to play all supported file types from the desktop as well as from the web.

## Stack

1. [Tauri](https://tauri.io/)
2. [Rust](https://www.rust-lang.org/)
3. [Svelte](https://svelte.dev/)

## To run

1. Clone the repo
2. Run

```bash
cd mediavibe
npm install
npm run tauri dev
```
